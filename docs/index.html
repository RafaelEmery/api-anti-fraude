Introdução

Seja bem vindo a documentação da API Anti Fraude! Para baixar, instalar e rodar a API basta ler as intruções no README.md. Caso você queira aprender como usar e testar, no README também terá uma seção que menciona o uso com o Postman

Aproveite a documentação, e caso tenha algum erro ou feedback, abra um Issue ou entre em contato com o desenvolvedor.

index

Endpoint que retorna todos as transações presentes em src/data/transactions.json

index(req: any, res: any, next: any): res
Parameters
req (any) requisição enviada
res (any) resposta que a função irá retornar
next (any) auxilia no tratamento de erros
Returns
res: todas as transações do arquivo json

transaction

Endpoint que recebe um array e retorna a pontuação do risco de fraude

transaction(req: any, res: any, next: any): res
Parameters
req (any) requisição que contém o array no corpo
res (any) resposta que a função irá retornar
next (any) auxilia no tratamento de erros
Returns
res: o id da transação e a pontuação (score)

getScore

Função auxiliar que implementa as regras e incrementa a pontuação (score) de acordo com as verificações

score(params: Object): number
Parameters
params (Object = {}) é um objeto contendo uma transação
Returns
number: score (total da pontuação) para o risco de fraude
